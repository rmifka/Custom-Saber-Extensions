What are custom saber extensions?


Custom Saber Extensions are an easy to install Addon for Unity. They will help you speed up your Saber Creation Process.

They are easy to use but for the sake of explaining im gonna explain every single feature there is. If you ever get lost while trying something you can always come back here.


Exporter Generator NEW

What is a Exporter Generator?

A Export Generator is a simple tool to create more Exporters. This allows you to have more than one Saber ready to Export.


How does it work?

It generates a Exporter with your chosen Name. In your Saber Export Menu you will be able to find a new Exporter now. It works exactly like the first one. All the scripts you need for a saber will be applied aswell.


Trail Point Generator NEW

What is a Trail Point Generator?

Its a simple tool to generate your Trail Top and Bottom.


How does it work?

You need to input your Saber Blade (The Gameobject that has the mesh!) and then it will calculate the Top Point of the trail and move the Bottom Point down depending on the width you chose.


Trail Preview Generator NEW

What is a Trail Preview Generator?

Its a tool to generate Trail Previews. It is like using a plane with a trail material on it.


How does it work?

You can chose the colors of your sabers, then you will need to input your LeftSaber Game Object and your RightSaber Game Object. After this it should automatically choose TrailTop and TrailBottom (If your Trail points do not have these names then you will need to input them yourself), and after all that you just need to set a material and a length. Now you can press generate Trail Planes and it should automatically place planes with the materials on them. (A known bug is if your Right Saber is on the left side it will generate the previews on the wrong position).


Saber Copier NEW

What is a Saber Copier?

Its a tool to copy a saber and change the material colors with Custom Color switches on to the selected color.


How does it work?

It copies every single Material to a new Folder called CopiedMaterials then to a subfolder with the Saber Name. It will only change Color if the Custom Color Switch on the material is enabled.

DISCLAIMER: IF A MATERIAL DOES NOT HAVE A CUSTOM COLOR SWITCH IT WILL NOT CHANGE THE COLOR


Saber Packer NEW

What is a Saber Packer

A saber packer is a simple tool I created to pack all of your Materials into a single folder.


How does it work?

First you will need to put your Exporter Game Object into the corresponding field. After this you give it a saber name and press Pack Materials. This will create a folder called Packed Sabers with a subfolder corresponding to the Saber Name. This subfolder will have two more subfolders called Left And Right Saber. In these you will be able to find the materials of your saber. The materials are also automatically applied to the Saber.

Disclaimer: This process works because one of the sabers is named LeftSaber if this does not exist it will not work.
Please make sure that at least one child is named LeftSaber. I do not know what happens if you don't have a LeftSaber Game Object.


Spin Animation Generator NEW

What is a Spin Animation Generator

A Spin Animation Generator is a simple tool to create Spin Animations for rings or text.


How does it work?

At first you will need to input the Game Object you want to spin. After this you will need to chose a duration (in seconds). Now you will need to decide if you want to use a parents rotation (All this does is if you have messed up rotations it will ignore those). The Next Step is to check the Axis you want to make Spin. At last you can chose if you want it to spin in the other direction. After this you just need to press Create Spin Animation.
